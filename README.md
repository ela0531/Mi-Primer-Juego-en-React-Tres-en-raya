## Breve descripción del juego
Este proyecto es un juego de Tres en Raya hecho en React. Tiene un modo para jugar entre dos personas y otro para jugar contra la máquina. El juego avisa cuando alguien gana, si hay empate, tiene botón para reiniciar y guarda el puntaje. También le agregué un estilo inspirado en los colores de la bandera dominicana y los nombres quisqueya vs Caribe.

## Qué aprendí
Con este proyecto aprendí a usar mejor React, sobre todo useState y useEffect ya que es mi primera vez trabajando con react y viendo como funciona. También aprendí cómo manejar turnos, guardar información con localStorage y crear la lógica para que el juego funcione correctamente.

## Qué parte fue más difícil
La parte más difícil fue hacer la lógica para detectar al ganador y programar el modo contra la máquina. También me costó al principio organizar algunas cosas en React, pero practicando pude entenderlo un poco mejor.

como ejecutar el proyecto 

Siguiendo estos pasos para correr la aplicacion en tu computador

1. Clona o descarga este repositorio.

2. Abre una terminal en la carpeta del proyecto.

3. Instala las dependencias:

npm install

4. Ejecuta la aplicación:

npm start
